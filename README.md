# StaffNow Dashboard

단기 스태프 채용 플랫폼 — 기업 운영자용 관리 대시보드

## 기술 스택

- React 18 + Vite
- Tailwind CSS v4
- React Router v7

## 실행 방법

```bash
npm install
npm run dev
```

## 포털 구성

| 역할 | 진입 경로 |
|------|-----------|
| 기업 담당자 | `/dashboard` |
| 개인 구직자 | `/individual` |
| 관리자 | `/admin` |

## 핵심 운영 플로우

기업 담당자가 공고 생성부터 정산까지 하나의 흐름으로 운영할 수 있습니다.

```
1. 공고 생성
   공고 관리 → 공고 생성 → 공고 상세
      |
      | "Shift 추가" (공고 자동 연결)
      ↓
2. Shift 생성
   Shift 생성 폼 (날짜·시간·필요인원 입력) → 생성 완료 시 Shift 상세로 이동
      |
      ↓
3. 지원자 선별 및 채용 확정
   Shift 상세 → 지원자 채용/거절 → "완료/확정" 버튼
      |
      | 확정 완료 시 계약 상태 페이지로 자동 이동
      ↓
4. 계약 확인
   계약 상태 → 계약서 상세 보기 (PDF 다운로드 UI)
      |
      | "근태 관리" 버튼
      ↓
5. 근태 관리
   출퇴근 기록 확인 (달력 + 정렬 가능 테이블)
      |
      | "정산 관리" 버튼
      ↓
6. 정산 승인
   미정산 내역 확인 → 초과 근무 캐핑/승인 → 개별/일괄 정산 완료
      |
      | 정산 완료 시 대시보드 미정산 건수 자동 감소
      ↓
완료
```

### 플로우 보조 연결

| 페이지 | 바로가기 |
|--------|----------|
| Shift 목록 (완료 탭) | 근태 관리, 계약 보기 버튼 |
| 초대/확정 (확정됨) | Shift 보기 링크 |
| 대시보드 미정산 카드 | 정산 관리 바로가기 |

## 주요 기능

- 공고 생성 및 관리 (계층형 지역 선택, 최저임금 검증)
- Shift 생성 및 인력 선별 (채용/거절, 별표 고정, 사이드 드로워)
- 채용 확정 데이터 localStorage 자동 저장
- 초대/근태/정산 관리
- 메시지 기능 (수정·삭제·파일 전송 UI, 미읽음 뱃지, 타이핑 인디케이터)
- 정산 초과 근무 캐핑 (유예 10분, 연장 승인 처리)
- 알림 시스템 (토스트)
